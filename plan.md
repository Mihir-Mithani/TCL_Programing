# TCL Learning Plan

## Phase 1: Fundamentals (Week 1-2)

- [x] TCL syntax and basic commands (`puts`, basic script structure)
- [x] Variables and data types (`set`, string/integer variables, `$` substitution)
- [x] Control structures — Conditionals (`if`, `if-else`, `if-elseif-else`)
- [x] Control structures — Loops (`while`, `for`, `break`, `continue`)
- [x] Loops — `foreach` loop over lists
- [x] Arrays — Indexed arrays (`set arr(index)`, `array names/size/get`, iteration)
- [x] Associative Arrays — String keys, `array names/set/get`, iteration
- [ ] Procedures (`proc`) and scoping
- [ ] Lists (`lindex`, `llength`, `lappend`, `lsort`, `lsearch`, `split`, `join`)
- [ ] String operations (`string length/index/range/compare/match/trim/replace`, `append`)
- [x] Arithmetic & Expressions (`expr`, math/comparison/logical operators, `incr`)

## Phase 2: Core Concepts (Week 2-3)

- [ ] File I/O operations (`open/close`, read/write/append, line-by-line, `file` commands)
- [ ] Regular expressions (`regexp`, `regsub`, wildcards, groups, VLSI use cases)
- [ ] Error handling (`catch`, `error`, `try/throw`, `errorInfo`, `errorCode`)
- [ ] Namespaces and packages (`source`, `package provide/require`, `namespace`)
- [ ] Event loop and `after` command
- [ ] Interpreters (`interp`)

## Phase 3: Advanced Topics (Week 3-4)

- [ ] Tk GUI toolkit basics
- [ ] Expect for automation
- [ ] Threading and concurrency
- [ ] C extensions / Tcl API
- [ ] Testing with tcltest
- [ ] Package management (teacup, tclpkg)

## Phase 4: Practice Projects

- [ ] Build a simple calculator
- [ ] Create a file organizer script
- [ ] Build a Tk-based TODO app
- [ ] Write a network client/server
- [ ] Create a test suite for a module
