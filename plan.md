# TCL Learning Plan

## Phase 1: Fundamentals (Week 1-2)
- [ ] TCL syntax and basic commands
- [ ] Variables and data types
- [ ] Control structures (if, while, for, foreach, switch)
- [ ] Procedures (proc) and scoping
- [ ] Lists and arrays
- [ ] String operations

## Phase 2: Core Concepts (Week 2-3)
- [ ] File I/O operations
- [ ] Regular expressions
- [ ] Error handling (catch, try, throw)
- [ ] Namespaces and packages
- [ ] Event loop and after command
- [ ] Interp (interpreters)

## Phase 3: Advanced Topics (Week 3-4)
- [ ] Tk GUI toolkit basics
- [ ] Expect for automation
- [ ] Threading and concurrency
- [ ] C extensions / Tcl API
- [ ] Testing with tcltest
- [ ] Package management (teacup, tclpkg)

## Phase 4: Practice Projects
- [ ] Build a simple calculator
- [ ] Create a file organizer script
- [ ] Build a Tk-based TODO app
- [ ] Write a network client/server
- [ ] Create a test suite for a module

## Resources
- [ ] Official TCL Tutorial: https://www.tcl.tk/man/tcl8.6/tutorial/tcltutorial.html
- [ ] TCL Wiki: https://wiki.tcl-lang.org/
- [ ] "Tcl and the Tk Toolkit" by Ousterhout
- [ ] Practice on Rosetta Code TCL examples
